Synthesis Techniques Using Basic Waveforms

Understanding how these waveforms are used in different synthesis methods is crucial for effective sound design:

Subtractive Synthesis

Primary Waveforms: Sawtooth, Square

Start with harmonically rich waveforms and remove frequencies using filters. Sawtooth waves are ideal because they contain many harmonics for the filter to sculpt.

Example: Sawtooth → Low‑pass filter → ADSR envelope = Classic analog lead sound

FM Synthesis

Primary Waveforms: Sine waves

Use sine waves as carriers and modulators. Different ratios create families of timbres from bell‑like to metallic.

Example: Sine carrier + Sine modulator at 2:1 ratio = Bell‑like tones

AM Synthesis

Any waveform can be amplitude‑modulated.

Modulate the amplitude of one waveform with another. Low rates yield tremolo; audio‑rate AM creates new sidebands.

Example: Sawtooth AM’d by triangle = Rhythmic, pulsing texture

Additive Synthesis

Primary Waveforms: Sine waves

Build complex sounds by summing many sines at different frequencies and amplitudes—each partial adds a piece of the tone.

Example: Fundamental + 3rd + 5th harmonic ≈ Square‑like tone

Pulse Width Modulation

Primary Waveforms: Square‑wave variants

Vary the pulse width of a square wave to animate its harmonic content—great for strings and choruses.

Example: PWM square wave = Classic analog string ensemble sound

Ring Modulation

Any two waveforms can be ring‑modulated.

Multiply two signals to generate sum/difference frequencies—often inharmonic and metallic.

Example: Sine × Square = Complex, inharmonic spectrum

Practical Sound Design Examples

Brass Sound

Start with: Sawtooth wave

Processing:

  • Low‑pass filter with moderate resonance
  • Filter envelope for brightness changes
  • Amplitude envelope with slow attack
  • Add vibrato with sine‑wave modulation

Classic Lead

Start with: Square or Sawtooth wave

Processing:

  • High‑pass filter to remove mud
  • Low‑pass filter for tone shaping
  • Fast attack, moderate decay envelope
  • Add portamento for smooth transitions

Bell Sound

Start with: Sine wave (FM)

Processing:

  • Modulator at ~3.5:1 ratio
  • High modulation index for brightness
  • Exponential decay envelope
  • Add reverb for space

Pad Sound

Start with: Multiple detuned saws

Processing:

  • Low‑pass filter with low cutoff
  • Very slow attack envelope
  • Add chorus/ensemble
  • Layer different octaves

Retro Game Sound

Start with: Square wave

Processing:

  • Fast attack, quick decay
  • Pitch envelope for “blip” effect
  • Minimal processing for clarity
  • Vary pulse width for articulation

Bass Sound

Start with: Sawtooth wave

Processing:

  • Low‑pass filter with envelope modulation
  • Sub‑oscillator (sine, −1 octave)
  • Compression for punch
  • Short decay for percussive attack

Pro Tips for Waveform Selection

  • Frequency matters: Higher notes reveal harmonics—audition across octaves.
  • Layering power: Combine waveforms (e.g., sine + saw) for warmth.
  • Phase relationships: Align phases when layering to avoid cancellations.
  • Context is key: A bright saw may be perfect in a dense mix but harsh soloed.
  • Modulation magic: Modulate parameters for evolving textures.

Choosing the Right Waveform

  • Pure, clean: Sine
  • Cut through a mix: Sawtooth or Square
  • Vintage/retro: Square
  • Complex filtering: Sawtooth
  • Smooth, mellow: Triangle

Mastering Waveform Fundamentals

Understanding these four basic waveforms is like learning the alphabet of electronic music—each one is a distinct letter you can combine into words and phrases.

In modern synthesis, these waves are starting points. The magic happens when you combine oscillators, filters, envelopes, and modulation to create motion and emotion.

Great sound design comes from knowing not just what each waveform sounds like, but why to choose it in context.
